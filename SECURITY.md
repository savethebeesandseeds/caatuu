# Caatuu security policy

Caatuu is development-preview software. No version is currently declared a
supported public beta or stable release.

Report a suspected vulnerability privately to `contact@waajacu.com`. Include
the affected version or source commit, reproduction steps, expected impact, and
the minimum evidence needed to investigate. Do not include unrelated personal
data, credentials, private keys, or destructive proof-of-concept activity.

Please do not publish an unresolved vulnerability before the maintainer has had
a reasonable opportunity to confirm and address it. Caatuu does not currently
promise a response-time or remediation-time service level.

Ordinary bugs that contain no sensitive information may be filed in the public
GitHub repository. The in-app diagnostic submission channel is disabled.
