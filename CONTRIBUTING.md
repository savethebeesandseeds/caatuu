# Contributing to Caatuu

Caatuu is currently founder-led and preparing its first properly governed beta.
External code, model, dataset, documentation, and artwork contributions are
temporarily paused while the project completes its ownership, licensing, and
inbound-contribution policy.

Please do not open a pull request or send contribution files yet. Unrequested
material will not be treated as accepted into the project.

Before contributions reopen, this document will state:

- the contribution workflow and review authority;
- the required certificate or contributor agreement;
- licensing rules for code, data, models, and artwork;
- provenance and attribution evidence required with each contribution;
- testing and release expectations; and
- the community conduct and moderation process.

This pause does not prevent people from using the existing support or security
channels once those channels are formally published.
