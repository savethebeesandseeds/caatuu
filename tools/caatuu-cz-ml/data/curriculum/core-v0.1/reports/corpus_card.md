# Caatuu Curriculum Core v0.1

Status: draft production-scale seed package.

This package contains 5000 language-neutral curriculum items and 5000 English realizations.

These rows are controlled Caatuu original seeds. They should still pass OpenAI curation and human spot review before training.

The current local Czech seed corpus remains useful as raw material, but it is not the canonical curriculum source.

Existing local counts at build time:

- Czech seed sentences: 4278
- Planet Word Net Czech sentence candidates: 6928

Next step: run the OpenAI batch curation pass on English realizations, review approvals/revisions, then adapt approved rows into Czech.
