# Caatuu demos

Large, isolated development experiments live here instead of inside a
production app's `static/assets` catalog. The Caatuu runtime serves this folder
at `/demos/`.

## Available project

- `world-movement/` - character locomotion, pose-authority, sprite-generation,
  and browser-animation laboratory.

Only the current reviewed experiments are kept in Git. Superseded generated
iterations and raw research inputs are reproducible working material, not
repository history.

Each project keeps its runtime files, documentation, licenses, and selected
reference assets together. Shared production assets may be referenced from
`/assets/`, but should not be copied into a demo.
