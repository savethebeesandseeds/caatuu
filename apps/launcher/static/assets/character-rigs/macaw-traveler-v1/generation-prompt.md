# Macaw traveler puppet-pack generation prompt

Use case: stylized-concept

Asset type: production 2D game-character cutout puppet parts sheet for a
browser RPG

Input images: Image 1 is the primary identity and rendering reference; Image 2
is a secondary costume, backpack, staff, and proportion reference

Primary request: preserve this exact elderly teal-and-white macaw traveler and
redraw him as one clean, exploded, articulated puppet-parts sheet. This is not
an animation frame sheet. It is a reusable skeletal-rig skin.

Scene/backdrop: perfectly flat, uniform solid `#ff00ff` chroma-key background;
no floor, shadows, gradients, checkerboard, texture, lighting variation, panel
borders, or labels

Style/medium: exactly match the references: polished hand-painted cartoon game
asset, crisp dark outlines, warm brown robe and backpack, cyan-blue hands,
orange bird feet, gray beard, teal tail feathers

Composition/framing: landscape sheet divided only by generous empty magenta
gutters into a strict 3-row by 4-column layout. Exactly one isolated component
centred in every cell. No component may touch or overlap another cell.

Required cells, left-to-right:

1. Row 1: complete head with beak and beard; torso with robe and backpack but
   no head, arms, legs, tail, or staff; tail-feather group; wooden walking staff.
2. Row 2: far upper sleeve/arm; far forearm with relaxed cyan hand; near upper
   sleeve/arm; near forearm with cyan hand shaped to grip the staff.
3. Row 3: far orange lower leg and foot; near orange lower leg and foot; spare
   cyan gripping hand; one fully assembled neutral macaw reference pose.

Rigging requirements: every detached limb must preserve the reference's exact
design and scale relationship. Add rounded hidden overlap under shoulders,
elbows, neck, tail root, and ankles so rotation cannot reveal gaps. Keep joint
ends clean and puppet-ready. The neutral assembled reference must show both feet
planted, body balanced, arms relaxed, and staff vertical.

Constraints: preserve character identity, facial features, age, proportions,
robe knot, rolled bedroll, backpack, palette, outline weight, and staff design.
Every requested part appears exactly once in its assigned cell. Crisp isolated
silhouettes with generous padding. Do not use `#ff00ff` anywhere in the artwork.

Avoid: sequential walking poses; duplicated full characters; missing hands;
fused limbs; cropped parts; perspective turnarounds; captions; arrows; grid
lines; cast shadows; scenery; checkerboards; watermark; text.
